# 🚀 Francisco Dias' 3D Portfolio

Welcome to my personal portfolio! This project showcases my skills, experiences, and projects as a **React Developer**, **Python Developer**, and more. Built with **React** and **Vite**, it’s designed to be fast, responsive, and visually engaging.

---

## 🌟 Features

- **Interactive 3D Models**: Explore my tech stack with dynamic 3D icons powered by `.glb` models.
- **Journey Highlights**: A timeline of my academic, professional, and volunteer experiences.
- **About Me**: Get to know my passions, values, and what drives me.
- **Social Links**: Connect with me on platforms like LinkedIn, Instagram, and more.
- **Modern Tech Stack**: Built with React, Vite, and modern JavaScript tools.

---

## 🛠️ Tech Stack

- **Frontend**: React, JavaScript, Vite
- **3D Models**: `.glb` files for interactive visuals
- **Styling**: CSS Modules / Tailwind (if applicable)
- **Deployment**: [Vercel](https://vercel.com/) / [Netlify](https://www.netlify.com/) (or your chosen platform)

## 📂 Project Structure

```plaintext
public/
├── images/             # Static images
├──── logos/            # Logos and icons
├──── textures/         # Textures for 3D models
├── models/             # 3D models (.glb files)

src/
├── components/         # Reusable React components
├──── models/           # 3D model components
├────── contact/        # Contact form component
├────── hero_models/    # Hero section with 3D models
├────── tech_logos/     # Tech logos component
├── constants/          # Static data (e.g., tech stack, journey cards)
├── sections/           # Components for different sections of the portfolio
├── App.jsx             # Main application component
├── index.css           # Global styles
└── main.jsx            # React DOM rendering
```

## 🧑‍💻 How It Works

This portfolio is designed to provide a seamless and interactive experience for visitors. Here's how it works:

1. **Dynamic Content**: The portfolio dynamically renders content using React components, ensuring a modular and reusable structure.
2. **3D Models**: `.glb` files are used to create interactive 3D visuals, adding a modern and engaging touch.
3. **Responsive Design**: The layout adapts to different screen sizes, ensuring a great experience on both desktop and mobile devices.
4. **Animations**: Smooth animations and hover effects enhance the user experience, making the portfolio visually appealing.
